# One "category | topic [| destination(s)]" per line. Lines starting with # are ignored.
# Save this as topics.txt and run:  node generate.js --topics topics.txt
#
# Format:
#   category | topic                      ← 2 fields, destinations auto-detected from title
#   category | topic | destination        ← 3 fields, destination explicit (highest priority)
#   category | topic | dest1, dest2       ← multiple destinations, comma-separated
#
# Destinations should match the `name` of an existing Destination in Strapi
# (case-insensitive). Slugs also work. Auto-detect runs first; explicit
# destinations override.
#
# The CLI will look up each category in Strapi (by slug or name), auto-create
# it if it doesn't exist, attach the article to it, and skip topics whose
# title (or derived slug) already exists in Strapi.
